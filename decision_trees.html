Decision Trees: Interview Q&A Style

Interviewer: Humne regression models dekhe. Decision Trees kaise alag hain? Yeh kaam kaise karte hain?

Decision Trees ekdum alag tareeke se kaam karte hain. Yeh classification aur regression dono ke liye use ho sakte hain. Yeh model data se simple IF-ELSE rules ka ek tree structure seekhte hain.

Socho ek flowchart jaisa: Aap ek question (feature) se start karte ho, uske answer ke basis par aap ek branch follow karte ho, phir agla question, aur aise hi karte hue aap final decision (leaf node) tak pahunch jaate ho.

Example: Kya bahar khelne jaayein?

  • Start -> Question: Outlook kaisa hai? (Sunny/Overcast/Rainy)
  • IF Sunny -> Question: Humidity कैसी hai? (High/Normal) -> IF High: No, IF Normal: Yes
  • IF Overcast -> Yes
  • IF Rainy -> Question: Hawa tez hai? (Strong/Weak) -> IF Strong: No, IF Weak: Yes

Model training ke time seekhta hai ki kaunse features (questions) aur unki kaunsi values (branches) data ko sabse acche se split karte hain (information gain/gini impurity jaise criteria use karke).

Interviewer: Yeh toh kaafi easy lag raha hai samajhne mein! Iske fayde aur nuksan kya hain? Aur yeh Random Forests kya hai?

Fayde (Pros):

  • Interpretability: Iska structure (flowchart) dekh kar samajhna aasan hai ki model decision kaise le raha hai.
  • Minimal Data Prep: Feature scaling ki zaroorat usually nahi padti.
  • Non-linear patterns handle kar sakta hai.

Nuksan (Cons):

  • Overfitting Prone: Agar tree bahut deep ban gaya, toh yeh training data ko overfit kar sakta hai.
  • Instability: Data mein chhote changes se tree structure kaafi badal sakta hai.

Random Forests: Yeh Decision Trees ki overfitting problem ko solve karne ka ek powerful tareeka hai. Ismein hum ek nahi, balki bahut saare Decision Trees banate hain (ek "forest"). Har tree ko training data ke random subset aur features ke random subset par train kiya jaata hai. Final prediction sabhi trees ki predictions ka average (regression) ya majority vote (classification) lekar kiya jaata hai. Isse model zyada robust aur accurate banta hai.

Example: Simple Decision Tree Diagram

Yeh ek simple decision tree hai predict karne ke liye ki koi person video game khareedega ya nahi (based on Age and if Student).

Age <= 30?
|-- YES --> Student?
|-- YES --> Prediction: Buy
|-- NO --> Prediction: Don't Buy
|-- NO --> Prediction: Buy (Assuming older people buy)
Aap dekh sakte hain ki kaise simple rules follow karke final prediction tak pahunch rahe hain. Random Forest mein aise bahut saare (thode alag-alag) trees honge.
