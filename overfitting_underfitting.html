Overfitting & Underfitting: Interview Q&A Style

Interviewer: Jab hum model train karte hain, toh kya woh hamesha acche se kaam karta hai naye data par?

Nahi, zaroori nahi. Do common problems aa sakti hain: Overfitting aur Underfitting.

Underfitting: Model itna simple hota hai ki woh training data ke underlying pattern ko theek se capture hi nahi kar paata. Woh training data par bhi accha perform nahi karta, aur naye (test) data par toh bilkul nahi.

  • Analogy: Exam ke liye bilkul padhai nahi ki, toh exam mein kuch nahi kar paoge.

Overfitting: Model itna complex ho jaata hai ki woh training data ke patterns ke saath-saath uske noise (random fluctuations) ko bhi 'ratt' leta hai. Woh training data par toh super-human performance deta hai, lekin naye (test) data par bahut kharab perform karta hai kyunki woh generalize nahi kar paata.

  • Analogy: Exam ke liye sirf specific questions rat liye. Agar exam mein thoda bhi alag question aaya, toh answer nahi kar paoge.

Hamara goal hai ek "Goldilocks" model banana - jo na underfit kare na overfit, balki data ke true underlying pattern ko seekhe aur naye data par accha generalize kare.

Interviewer: Toh pata kaise chalta hai ki model overfit ya underfit kar raha hai? Aur isse theek kaise karte hain?

Hum model ki performance ko training data aur ek alag test data (jo model ne training ke time nahi dekha) dono par check karte hain:

  • Underfitting: Performance training aur test dono par kharab hoti hai.
  • Overfitting: Performance training par bahut acchi hoti hai, lekin test par kaafi gir jaati hai.
  • Good Fit: Performance training aur test dono par acchi hoti hai, aur test performance training se thodi hi kam hoti hai.

Solutions:

  • Underfitting ke liye: Model ko complex karo (e.g., zyada features use karo, complex algorithm try karo), zyada der train karo.
  • Overfitting ke liye: Model ko simplify karo (e.g., kam features use karo), aur data collect karo, Regularization techniques use karo (yeh model complexity ko penalize karti hain), Cross-validation use karo.
