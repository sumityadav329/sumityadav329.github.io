CNNs: Interview Q&A Style

Interviewer: Neural Networks toh theek hain, par images jaise data ke liye (jahan pixels ka spatial arrangement important hai) kya normal NN کافی hai? CNNs kaise different hain?

Normal NNs (Dense/Fully Connected) image data ke liye akele efficient nahi hote. Agar aap ek 100x100 pixel image ko flatten karke input layer mein daalo, toh 10,000 input neurons ho jayenge, aur hidden layer ke saath connections bahut zyada ho jayenge (millions of weights!). Yeh computationally expensive hai aur spatial information (pixels ki relative position) loose ho jaati hai.

Convolutional Neural Networks (CNNs) specifically image (aur dusre grid-like data jaise audio spectrograms) ko process karne ke liye design kiye gaye hain. Yeh spatial hierarchies ko preserve karte hain.

CNNs kuch special layers use karte hain:

  • Convolutional Layer: Yeh core building block hai. Ismein small filters (kernels) input image par slide karte hain, element-wise multiplication aur sum karte hain (convolution operation). Har filter specific feature detect karna seekhta hai (jaise edges, corners, textures). Ek hi filter puri image par use hota hai (parameter sharing), jo NNs se bahut efficient hai.
  • Activation Layer (ReLU): Convolution layer ke output mein non-linearity add karta hai.
  • Pooling Layer (e.g., Max Pooling): Feature map ko downsample karta hai (size reduce karta hai). Yeh computation kam karta hai aur model ko object ki exact position ke prati thoda robust banata hai (translation invariance). Max pooling usually ek small window mein se maximum value leta hai.

Yeh layers (Conv -> ReLU -> Pool) multiple baar stack ki jaati hain hierarchical features seekhne ke liye (simple edges -> shapes -> object parts -> objects).

Interviewer: Toh pehle CNN layers features extract karti hain, aur phir final decision kaise hota hai?

CNN architecture ke end mein, jo high-level feature maps aate hain multiple Conv/Pool layers ke baad, unhe flatten kiya jaata hai (ek single long vector mein convert kiya jaata hai).

Is flattened vector ko phir ek standard Fully Connected Neural Network (jaise humne pehle dekha) mein input ki tarah diya jaata hai. Yeh fully connected layers in high-level features ko combine karke final classification (e.g., Cat/Dog/Car) ya regression task perform karti hain. Output layer mein usually Softmax (multi-class classification) ya Sigmoid (binary) use hota hai.
